# BMI Calculator

A responsive and user-friendly Body Mass Index (BMI) calculator with both metric and imperial unit support, featuring light/dark mode toggle.

## Features

- Calculate BMI using both metric (kg/cm) and imperial (lbs/inches) units
- Real-time BMI category classification (Underweight, Normal, Overweight, Obese)
- Personalized health recommendations based on BMI result
- Light/Dark mode toggle with localStorage persistence
- Responsive design that works on all device sizes
- Clean and intuitive user interface

## How to Use

1. Select your preferred unit system (Metric or Imperial)
2. Enter your weight and height values
3. Click "Calculate BMI" to see your results
4. View your BMI value, category, and personalized health recommendations

## BMI Categories

- **Underweight:** BMI less than 18.5
- **Normal weight:** BMI between 18.5 and 24.9
- **Overweight:** BMI between 25 and 29.9
- **Obese:** BMI 30 or greater

## Technologies Used

- HTML5

- JavaScript (ES6+)
