# Quickstart Guide: BMI Calculator

## Getting Started

The BMI calculator is a static web application that requires no installation or setup. Simply open the `index.html` file in any modern web browser to start using the calculator.

## How to Use

1. Open `index.html` in your web browser
2. Select your preferred measurement system (Metric or Imperial)
3. Enter your weight in the appropriate units
4. Enter your height in the appropriate units
5. Click "Calculate BMI" button
6. View your BMI result, category, and health recommendation

## Development

To modify the calculator:

1. Edit `index.html` to change the structure
2. Edit `style.css` to change the appearance
3. Edit `script.js` to change the functionality
4. Open `index.html` in a browser to test your changes

## File Structure

- `index.html`: Main HTML structure and form elements
- `style.css`: Styling, layout, and responsive design
- `script.js`: BMI calculation logic and UI interactions

## Browser Compatibility

The calculator works in all modern browsers (Chrome, Firefox, Safari, Edge) that support HTML5, CSS3, and ES6 JavaScript features.
